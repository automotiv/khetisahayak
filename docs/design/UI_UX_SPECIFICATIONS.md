## Design Principles

### 1. Farmer-First Design
- **Simple & Intuitive**: Designed for users with varying digital literacy
- **Visual Communication**: Icons and images over text where possible
- **Large Touch Targets**: Minimum 48x48dp for all interactive elements
- **Clear Hierarchy**: Important actions prominently displayed

### 2. Agricultural Context
- **Nature-Inspired Colors**: Greens, earth tones, sky blues
- **Contextual Imagery**: Real farming scenarios, not stock photos
- **Local Relevance**: Indian agricultural context in all visuals

### 3. Offline-First Mindset
- **Clear Status Indicators**: Always show connectivity status
- **Graceful Degradation**: Features work offline where possible
- **Sync Feedback**: Clear indication of pending/synced data

### 4. Multilingual Ready
- **Text Expansion**: Design for 30% text expansion
- **RTL Support**: Layout works for Urdu
- **Script Variety**: Support for Devanagari, Tamil, Telugu, etc.

- Cards in grid (2 columns)

## Accessibility

### WCAG 2.1 AA Compliance

1. **Color Contrast**
   - Text: Minimum 4.5:1 ratio
   - Large text: Minimum 3:1 ratio
   - Interactive elements: Minimum 3:1 ratio

2. **Touch Targets**
   - Minimum size: 48dp Ã— 48dp
   - Spacing between targets: 8dp minimum

3. **Text Scaling**
   - Support up to 200% text scaling
   - No horizontal scrolling at 200%

4. **Screen Reader Support**
   - All images have alt text
   - Form fields have labels
   - Buttons have descriptive text

5. **Focus Indicators**
   - Visible focus ring on all interactive elements
   - Focus order follows visual order

### Voice Navigation

```dart
// Semantic labels for screen readers
Semantics(
  label: 'Scan crop for disease detection',
  button: true,
  child: ScanButton(),
)
```

## Animation Guidelines

### Timing

| Animation Type | Duration | Easing |
|----------------|----------|--------|
| Micro-interactions | 100-200ms | ease-out |
| Page transitions | 300ms | ease-in-out |
| Loading states | 500ms+ | linear |
| Emphasis | 200-400ms | ease-out |

### Motion Principles

1. **Purposeful**: Every animation serves a purpose
2. **Quick**: Animations should feel snappy
3. **Natural**: Use physics-based easing
4. **Consistent**: Same animation for same action

### Common Animations

```dart
// Card tap feedback
ScaleTransition(
  scale: Tween(begin: 1.0, end: 0.95).animate(controller),
  child: Card(),
)

// Page transition
SlideTransition(
  position: Tween(
    begin: Offset(1.0, 0.0),
    end: Offset.zero,
  ).animate(CurvedAnimation(
    parent: controller,
    curve: Curves.easeInOut,
  )),
  child: NewPage(),
)

// Loading shimmer
Shimmer.fromColors(
  baseColor: Colors.grey[300]!,
  highlightColor: Colors.grey[100]!,
  child: Container(),
)
```

## Design Assets

### Required Assets

1. **App Icons**
   - Android: mdpi, hdpi, xhdpi, xxhdpi, xxxhdpi
   - iOS: 1x, 2x, 3x
   - Web: favicon, apple-touch-icon

2. **Splash Screen**
   - Logo centered
   - Gradient background
   - Loading indicator

3. **Empty States**
   - No products illustration
   - No orders illustration
   - No internet illustration
   - No results illustration

4. **Onboarding**
   - 4-5 screens with illustrations
   - Feature highlights
   - Language selection

5. **Crop Icons**
   - 20+ crop type icons
   - Consistent style
   - Multiple sizes (24dp, 48dp, 96dp)
