# System Architecture

## Overview

Kheti Sahayak follows a modern microservices architecture.

## Components

1. **Frontend**
   - Mobile App (React Native)
   - Web Portal (React)

2. **Backend**
   - API Gateway
   - Authentication Service
   - Data Processing Service

3. **Database**
   - PostgreSQL for relational data
   - MongoDB for documents
   - Redis for caching

4. **External Services**
   - Payment Gateway
   - SMS/Email Service
   - Weather API
